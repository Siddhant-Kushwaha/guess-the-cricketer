How to play

Guess the Cricketer is a daily cricket guessing game. Each day there's one mystery cricketer and you get eight guesses. Type any player and the game grades seven clues — country, role, batting hand, bowling style, debut year, IPL and World Cup — with green, yellow and grey tiles (see “what do the colours mean?” above the board). A new cricketer drops every midnight IST.

Modes

IPL mode — the mystery player is anyone who's played the IPL, so the names are recognisable. Legends mode — the answer is only an all-time great. In both modes you can still type and guess any cricketer to get clues. Unlimited practice serves a fresh random player and never touches your streak.

FAQ

Is it free? Yes — no login, no downloads, no real money. When does it reset? Midnight India Standard Time. Where does the data come from? Player roles follow ESPNcricinfo; facts are checked against public sources. Spot something wrong? Use the “report” link after you solve.
